## 5. 部署与环境

### 5.1 开发环境

- **端口**：3000（Vite dev server）
- **Mock 模式**：`VITE_USE_MOCK=true`（默认）
- **启动命令**：`npm run dev`

### 5.2 生产环境

- **构建命令**：`npm run build`
- **产物目录**：`dist/`
- **预览命令**：`npm run preview`
- **Mock 模式**：`VITE_USE_MOCK=false`
- **后端代理**：`/api` → `http://localhost:8080`（联调时可修改）

配置文件：
- `.env.development`：开发环境变量
- `.env.production`：生产环境变量
- `vite.config.js`：构建配置与 Mock 开关

## 7. 版本管理建议

- **前端依赖更新**：谨慎升级 Element Plus 大版本，可能有 breaking changes。
- **后端版本**：Spring Boot 3.x 要求 Java 17+，升级需注意兼容性。
- **数据库驱动**：待补充（需确认后端使用的 MySQL JDBC 驱动版本）。
