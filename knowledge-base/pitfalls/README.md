## 容易踩的点（预警，非问题）

这些不是 bug，但新人常困惑，先记下来：

- **请求已自动拆包**：业务层从 `api/*.js` 拿到的就是 `data`，不要再 `.data.data`。见 `src/api/request.js:32`。
- **业务错误已自动弹错**：`code !== 0` 时 `request.js` 已 `ElMessage.error`，页面 `catch` 不必重复弹，只需复位 UI。
- **Mock / 真实后端靠环境变量切**：现象「改了后端没生效」多半是 `VITE_USE_MOCK` 还是 `true`。见 `.env.development`。
