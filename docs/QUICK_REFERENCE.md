## 🔄 用户操作流程

### 添加关键词
1. 切换到"高危关键词配置" tab
2. 点击 [添加关键词] 按钮
3. 填写表单：
   - 关键词内容 *
   - 触发级别 *
   - 所属类别 *
   - 说明（可选）
   - 启用状态（默认开启）
4. 点击 [确定] 保存

### 编辑关键词
1. 在关键词列表中找到目标关键词
2. 点击 [编辑] 按钮
3. 修改表单内容
4. 点击 [确定] 保存

### 启用/禁用关键词
1. 直接点击表格中的状态开关
2. 系统自动保存

### 删除关键词
1. 点击 [删除] 按钮
2. 确认删除弹窗
3. 点击 [确定] 删除
