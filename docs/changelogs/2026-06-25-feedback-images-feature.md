# 用户反馈图片展示功能 - 更新日志

**日期**: 2026-06-25  
**功能**: 用户反馈列表新增图片展示与预览  
**影响范围**: 用户反馈模块

---

## 功能概述

为用户反馈列表新增了图片展示功能，支持：
1. 表格中展示缩略图（最多显示3张，超出显示 +N）
2. 点击缩略图预览大图
3. 大图预览支持左右切换查看所有图片

---

## 修改清单

### 1. Mock 数据更新

**文件**: `mock/index.js`

为 `feedbackRecords` 数组中的每条记录新增 `images` 字段（数组类型），存储图片 URL 列表。

示例数据：
```javascript
{
  id: 1,
  // ... 其他字段
  images: [
    'https://picsum.photos/400/300?random=1',
    'https://picsum.photos/400/300?random=2'
  ]
}
```

**测试数据分布**：
- 有 2 张图片的反馈：1 条（ID: 1）
- 有 1 张图片的反馈：2 条（ID: 4, 8）
- 有 3 张图片的反馈：1 条（ID: 6）
- 无图片的反馈：6 条

---

### 2. 页面组件更新

**文件**: `src/views/FeedbackRecords.vue`

#### 2.1 表格新增图片列

在"问题描述"列之后新增"图片"列（宽度 180px）：

**功能**：
- 显示前3张图片的缩略图（50x50px）
- 第3张图片如果有更多图片，显示 `+N` 遮罩
- 无图片时显示 `-`
- 点击缩略图打开预览对话框

#### 2.2 图片预览对话框

**功能**：
- 居中展示大图（最大高度 70vh）
- 顶部显示当前图片序号（如 "1 / 3"）
- 底部提供"上一张"/"下一张"按钮
- 支持键盘左右箭头切换（可选，未实现）

**响应式宽度**：
- 宽屏（>1200px）：70%
- 中等屏（768-1200px）：85%
- 小屏（<768px）：95%

#### 2.3 新增状态与函数

**状态**：
```javascript
const imagePreviewVisible = ref(false)      // 预览对话框显示状态
const previewImageList = ref([])            // 当前预览的图片列表
const currentImageIndex = ref(0)            // 当前预览图片索引
const imagePreviewWidth = computed(...)     // 响应式对话框宽度
```

**函数**：
```javascript
function previewImages(images, startIndex = 0) {
  // 打开预览对话框，从指定索引开始
  previewImageList.value = images
  currentImageIndex.value = startIndex
  imagePreviewVisible.value = true
}
```

## 使用说明

### 用户操作流程

1. 进入"用户反馈"页面
2. 在表格中查看"图片"列
3. 点击任意缩略图
4. 在预览对话框中查看大图
5. 使用"上一张"/"下一张"按钮切换图片
6. 点击对话框外部或关闭按钮退出预览

### 开发者扩展

如需修改图片来源（从真实后端 API 获取），只需确保：
1. API 返回的反馈记录包含 `images` 字段（字符串数组）
2. 图片 URL 可直接访问（或已处理跨域）

## 技术细节

### 图片加载优化

- 缩略图使用 `object-fit: cover` 裁剪适配
- 大图使用 `object-fit: contain` 完整展示
- 建议后端返回缩略图 URL 和大图 URL 分别优化（本次未实现）

### 性能考虑

- 表格每行最多渲染 3 张缩略图
- 预览对话框只加载当前图片
- 未实现图片预加载（可优化）

### 待优化项

1. **键盘支持**：添加左右箭头键切换图片
2. **懒加载**：表格滚动时懒加载缩略图
3. **图片压缩**：后端提供不同尺寸的图片 URL
4. **缩放功能**：预览对话框支持鼠标滚轮缩放
5. **拖拽关闭**：预览对话框支持下拉关闭手势
